Importante: Clique no botão "?" azul para ver as instruções completas de como criar o Apps Script.

Como Configurar o Google Sheets

Passo a passo completo

1

Criar a Planilha no Google Sheets

Acesse sheets.google.com e crie uma planilha em branco.

Não precisa criar cabeçalhos. O código cria automaticamente!

2

Abrir o Editor de Scripts

Na planilha, clique em Extensões → Apps Script

3

Colar o Código

  1. Apague todo o código padrão (função myFunction)
  2. Clique no botão "Copiar Código" abaixo
  3. Cole no editor do Apps Script
  4. Salve (Ctrl+S)
4

Publicar como Web App

  1. No Apps Script, clique em Implantar → Nova implantação
  2. Clique na engrenagem ⚙️ e selecione App da Web
  3. Descrição: Container API
  4. Executar como: Eu
  5. Quem tem acesso: Qualquer pessoa
  6. Clique em Implantar
  7. Autorize: clique em "Revisar permissões" → "Avançado" → "Ir para..." → "Permitir"
5

Copiar a URL e Conectar

  1. Copie a URL do Web App que aparece após implantable
  2. Volte para esta página e clique na engrenagem ⚙️
  3. Cole a URL no campo
  4. Clique em "Salvar e Conectar"

A bolinha do status deve ficar verde (Online)

Dicas

  • • Modo Offline: Sem Google Sheets, os dados ficam apenas no navegador
  • • Sincronização: Todos com a mesma URL veem os mesmos dados
